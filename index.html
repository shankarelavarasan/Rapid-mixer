Rapid AI Assistant

A Node.js backend for AI-powered content generation using Google's Gemini API

About the Project

Rapid AI Assistant is a Node.js backend application that leverages Google's Gemini API to provide AI-powered content generation. The application offers template-based processing and robust error handling to ensure reliable operation.

View on GitHub

Key Features

  • Integration with Google's Gemini API for AI content generation
  • Template-based processing for consistent outputs
  • Robust error handling with custom error types
  • Efficient file processing with caching
  • RESTful API endpoints for easy integration

Recent Improvements

Enhanced Error Handling

Implemented a comprehensive error handling strategy with custom error types and centralized error handling functions.

Improved Code Organization

Refactored the codebase into service modules for better separation of concerns and maintainability.

Comprehensive Testing

Added extensive test coverage for critical components including error utilities, file processing, template operations, and Gemini API interactions.

API Endpoints

  • /api/ask-gemini - Send prompts to the Gemini API
  • /api/templates - List available templates
  • /api/templates/:name - Get a specific template
